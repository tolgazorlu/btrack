# btrack

**btrack** is a CLI time tracker designed for developers who live in the terminal. It's fast, keyboard-driven, and integrates directly into your dev workflow — no browser tab required.

## Key Features

- **Session tracking** — Start, note, and stop sessions with a single command
- **Said vs shipped** — `btrack shipped` lines up sessions against `git log` so you can see what actually landed
- **Projects** — Organize work by project with per-project configuration via `.btrack` files
- **Notes** — Attach notes to running or past sessions without interrupting flow
- **Pomodoro** — Built-in Pomodoro timer that auto-tags sessions
- **AI summaries** — Generate standups and productivity insights from your sessions
- **Invoicing** — Generate invoice reports from tracked time with per-project rates
- **Google Calendar sync** — Push sessions to Google Calendar or pull events as sessions
- **Shell prompt** — Display active session info in your terminal prompt via Starship or PS1
- **Export** — Export sessions to CSV or JSON for use in other tools
- **GitHub integration** — Sync activity with GitHub commits and PRs

## Getting Started

Install btrack, then start your first session:

```bash
btrack s "fix login bug"
```
