# Shell Prompt Integration

Display your active btrack session directly in your shell prompt so you always know what you're tracking. Outputs nothing when no session is active — zero noise when you're idle.

## Quick Setup

Run the setup command for your shell to get a ready-to-paste snippet:

```bash
btrack shell zsh    # Zsh (RPROMPT)
btrack shell bash   # Bash (PS1)
btrack shell fish   # Fish (right prompt)
```

## Manual Setup

### Zsh

Add to `~/.zshrc`:

```zsh
btrack_prompt() { btrack prompt 2>/dev/null; }
RPROMPT='$(btrack_prompt)'
```

The session appears on the right side of your prompt. Replace `RPROMPT` with `PROMPT` to put it on the left.

### Bash

Add to `~/.bashrc`:

```bash
btrack_prompt() {
  local s=$(btrack prompt 2>/dev/null)
  [ -n "$s" ] && echo " $s"
}
PS1='\u@\h \w$(btrack_prompt) \$ '
```

### Fish

Create `~/.config/fish/functions/fish_right_prompt.fish`:

```fish
function fish_right_prompt
  btrack prompt 2>/dev/null
end
```

### Starship

Add to `~/.config/starship.toml`:

```toml
[custom.btrack]
command = "btrack prompt --format starship"
when    = "btrack prompt"
format  = "[$output]($style) "
style   = "blue"
```

Then add `custom.btrack` to your `format` string:

```toml
format = """
$username$directory$git_branch$custom.btrack$character
"""
```

## Output Formats

| Flag | Output |
|------|--------|
| _(none)_ | `fix login bug · 23m` |
| `--format starship` | JSON for Starship custom module |
| `--format json` | Machine-readable JSON |
| `--max-len N` | Truncate task name to N chars (default 30) |
