# AI Integration

btrack integrates with leading AI providers to turn your session history into standups, summaries, and productivity dashboards.

## Setup

```bash
btrack ai setup
```

Interactive setup that configures your AI provider and API key. Supported providers:

| Provider | Models |
|----------|--------|
| OpenAI | GPT-4o, GPT-4 Turbo |
| Anthropic | Claude Sonnet, Claude Haiku |
| Google | Gemini Pro, Gemini Flash |

Your API key is stored locally in `~/.config/btrack/config.yaml` and never sent to btrack servers.

## Standup

```bash
btrack standup
```

Generates a standup in **Yesterday / Today / Blockers** format from your tracked sessions. Defaults to yesterday — run it in the morning before your standup meeting.

```
Yesterday:
  • Fixed JWT auth bug (2h)
  • Wrote unit tests for auth module (1h)

Today:
  • PR review, deploy to staging

Blockers:
  • None
```

| Flag | Description |
|------|-------------|
| _(none)_ | Yesterday's sessions (default) |
| `--today` | Today's sessions so far |
| `-d N`, `--days N` | Last N days |

If GitHub is connected, commits and PRs are included automatically.

## AI Chat

```bash
btrack ai
```

Opens an interactive chat session with context from your recent btrack sessions. Ask questions like:

- _"What did I work on yesterday?"_
- _"How much time did I spend on auth this week?"_
- _"Write a PR description for my last session"_

## AI Summarize

```bash
btrack ai sum         # today's sessions
btrack ai sum --days 3
```

An alternative to `btrack standup` — generates a free-form summary instead of the structured Yesterday/Today/Blockers format.

## Productivity Insights

```bash
btrack ai ins
btrack ai ins --days 14
btrack ai ins --no-ai   # stats only, no AI key needed
```

Displays a dashboard analyzing your sessions:
- Peak productivity hours
- Most worked-on projects and tags
- Focus vs. context-switching patterns
- AI analysis with actionable suggestions

## GitHub Enrichment

Connect GitHub to include real commits and PRs in standups and summaries:

```bash
btrack github connect
```

Once connected, `btrack standup` and `btrack ai sum` automatically pull in your GitHub activity for the period.
